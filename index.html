论欲望

众所周知人类拥有各种各样的欲望，有食欲、性欲等等。其中分为必须欲望和非必须欲望。必须欲望顾名思义就是必须要的欲望，必须欲望如食欲，如果你不吃饭的话就会饿死。
非必须欲望就是人类有这个欲望很想得到，但是如果没有得到的话也不会死。
人类的欲望是永无止境的，顾所以要对欲望要加以控制。两种欲望都要加已控制，比较要更加重要的是后者非必须欲望。对于欲望的控制有几种方法，第一种也是最重要的一种
就是用其他爱好（欲望）分解你想要控制的欲望。还有一种就是少提起你想要控制的欲望，无论以任何形式提起一旦进入就如同决堤的洪水你是无法阻挡的。

linux常见命令

2019-08-13
技术
  • linux

文件命令:

  1. cat 显示文本文件内容

  2. echo 生成一个带内容文件

  3. mkdir 创建目录

  4. rm 删除文件
    -f 强制删除文件或目录
    -r 同时删除该目录下的所有文件

  5. mv 移动文件或目录

  6. find 在文件系统中查找指定的文件
    -name 文件名

  7. ln 建立软链接

  8. tail 显示文件结尾内容
    -f 跟踪输出

  9. grep搜索字符串

压缩命令

  1. tar tar [-cxzjvf] 压缩打包文档的名称 欲打包目录
    -c 建立一个归档文件的参数指令
    -x 解开一个归档文件的参数指令
    -z 是否需要用 gzip 压缩（常用）
    -j 是否需要用 bzip2 压缩
    -v 压缩的过程中显示文件
    -f 使用档名，在 f 之后要立即接档名
    -tf 查看归档文件里面的文件

管理命令

  1. top 显示当前系统中耗费资源最多的进程

  2. ps -ef显示瞬间的进程状态

  3. df 显示文件系统磁盘空间的使用情况 -h

  4. du 显示指定的文件（目录）已使用的磁盘空间的总量 -h

  5. kill 杀死一个进程 -9

  6. chmod 改变访问权限 chmod [who] [+ | - | =] [mode] 文件名
    who u：用户 user
    g：用户组 group
    o：表示其他用户
    a：表示所有用户是系统默认的

    mode 表示可执行的权限，可以是 r、w、x

  7. chown是 change owner 的意思，主要作用就是改变文件或者目录所有者，所有者包含用户和用户组
    chown [-R] 用户名称 文件或者目录
    chown [-R] 用户名称 用户组名称 文件或目录
    -R：进行递归式的权限更改，将目录下的所有文件、子目录更新为指定用户组权限

  8. 关机 shutdown -h now
    重启
    reboot
    shutdown -r now

  9. 列出所有的端口

  10. 比如查看80端口的使用的情况。
